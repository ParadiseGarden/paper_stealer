============================================================
  偷卷模拟器 PaperStealer — 完整说明文档
============================================================

一款校园题材的扫雷游戏。玩家扮演偷卷人，在档案室的保险柜
中收集试卷，避开老师和炸弹。包含成就系统、隐藏彩蛋、歌
词本、双模式切换等丰富功能。


------------------------------------------------------------
  快速开始
------------------------------------------------------------

  1. 打开 index.html（Chrome / Edge 推荐）
  2. 首页左侧选择难度（初中 / 高中 / 大学）
  3. 可选切换模式：📖 经典 / ⚡ 瑾耀护航
  4. 点击门上按钮进入游戏
  5. 左键翻柜子 / 右键上锁 / 点击数字快速展开周围


------------------------------------------------------------
  游戏模式
------------------------------------------------------------

  📖 经典模式（默认）
     - 标准扫雷规则，无额外技能
     - 成绩计入纪录榜

  ⚡ 瑾耀护航模式
     - 三个作弊技能可用
     - 踩雷后可使用「瑾耀护航」回退一步
     - 成绩不计入纪录榜

  两种模式互斥，在首页左侧勾选。


------------------------------------------------------------
  作弊技能
------------------------------------------------------------

  🔗 人脉网络 — 点击一格，翻开整行+整列，地雷自动标记
  📉 炒币爆仓 — 点击一格，翻开周围区域
                  （初中 3×3 / 高中 5×5 / 大学 7×7）
  👁 灰色资产 — 悬停柜门透视内部（需先首次点击解锁）

  悬停时高亮显示技能范围：金色 = 人脉网络，橙红 = 炒币爆仓。

------------------------------------------------------------
  工具栏（左上角）
------------------------------------------------------------

  📖  游戏说明
  ⚙️  设置（音量 / 灯光 / 歌词 / 返回首页）
  🏅  成就（连点 5 次可重置）
  🏆  最佳纪录


------------------------------------------------------------
  设置面板
------------------------------------------------------------

  🎵 音乐音量 — 滑块 0–100，默认 80
  🔊 音效音量 — 滑块 0–100，默认 80
  💡 开关灯   — 暗夜模式切换（游戏中开灯会触发失败）
  🎤 歌词     — 9 页歌词本，左右翻页
  🏠 返回首页


------------------------------------------------------------
  暗夜模式
------------------------------------------------------------

  首页自由开关，配色即时切换。
  游戏内强制暗夜，中途开灯会弹出警告，确认后直接失败。
  开灯导致的失败不可使用瑾耀护航恢复。


------------------------------------------------------------
  成就系统（🏅）
------------------------------------------------------------

  共 25 个成就，分 5 类：

  【通关】初一新生 / 高一新生 / 保研成功 / 课间十分钟 /
          午休狂奔 / 考前五分钟
  【技艺】关系户 / 散户收割机 / 偷窥狂 / 三好学生 /
          学霸本霸 / 专注力 MAX
  【挑战】秒躺 / 门后是谁
  【积累】刷题狂魔 / 题库终结者 / 人形打印机 /
          考场老手 / 屡败屡战 / 挑灯夜读
  【隐藏】不知死活 / 别敲了 / 黑板报读者 / 后台过硬 /
          歌词鉴赏家 / 破坏公物

  积累类成就显示进度条。隐藏成就锁定时显示「???」。
  达成时弹出金色提示。所有数据保存在浏览器本地存储。


------------------------------------------------------------
  彩蛋
------------------------------------------------------------

  👤 门后人物 — 连点窗户切换人物形象
                2 秒内连点 5 次触发彩蛋形象 + 震屏

  🕐 挂钟掉落 — 2 秒内连点挂钟 5 次
                挂钟旋转坠落 → 门后升起炸弹 → 破坏公物弹窗

  📋 黑板报   — 点击切换内容，共 3 页

------------------------------------------------------------
  自定义
------------------------------------------------------------

  素材开关（index.html 内）：
    const USE_CUSTOM_IMAGES = true;
    const USE_CUSTOM_AUDIO  = true;

  弹窗文案：搜索 DIFFICULTIES 修改各难度的 winDialog / loseDialog
  护航文案：搜索 CHEAT_PROTECT_DIALOG
  歌词内容：搜索 LYRICS_DATA（9 页 × 4 行）
  黑板报内容：搜索 BLACKBOARD_SETS（3 页 × 5 行）
  成就定义：搜索 ACHIEVEMENTS（25 个）

  加载失败的素材自动降级为 CSS / Emoji。
  控制台可调用 __resetAch() 重置所有成就。
